# E-Commerce Order Prediction using Machine Learning

This project predicts the outcome of an e-commerce order using Machine Learning.

Possible predictions:
- Delivered
- Cancelled
- Return to Origin (RTO)

## Model Used
Random Forest Classifier

## Features Used
- Customer State
- Quantity
- Price
- Month

## Technologies
- Python
- Pandas
- Scikit-learn
- Joblib

## Files
train_model.py – trains the ML model  
predict.py – prediction system  
model.pkl – saved ML model
